## 요약  

화물 운송이 화물차에 쏠려 있는 수단 편중으로 화물차로 인한 온실가스 98.1백만 톤이 수송부문에서 배출되고 있으며, 도로화물이 배출하는 온실가스 비중이 95.8%를 차지한다. 또한 **도로이동오염원 중에서 화물차가 차지하는 비중은 약 4분의 3가량의 도로 미세먼지**를 발생시킨다. 따라서 기존 시〮군〮구 단위의 분석이 아닌 읍/면/동 단위의 미시적 화물차 통행량 분석을 진행함으로써 권역별 고물동량 읍·면·동을 IC 접근성으로 정량평가한 결과 **각 권역 수도권, 영남권, 호남권, 층정권에서 각각 포승읍, 선암동, 하남동, 대소면이 1순위 거점 후보로 채택**되었다. 해당 지역에 물류 거점 입지로 선정하므로써, 공공 및 환경 측면에서 불필요한 도심 진입과 공차 운행을 줄여 도로 혼잡과 탄소 배출 저감에 기여하고자 한다.

## 1. 문제정의  
우리나라는 경유 화물차 운행에 절대적으로 의존하는 도로화물 수송체계로 2018년 기준 우리나라 국가 온실가스 배출량은 727.6백만 톤이며, 이 중 약 13.5%인 98.1백만 톤이 수송부문에서 배출되고 있는데, 이 때 **도로화물이 배출하는 온실가스 비중이 95.8%라고** 밝혀졌다.[^1]

2019년 기준 도로이동오염원 중에서 화물차가 차지하는 비중은 초미세먼지가 75.5%, 질소산화물이 52.5%로 **화물차가 약 4분의 3가량의 도로 미세먼지를 발생**시킨다는 결과를 볼 수 있으며,
이동량 측면에서 볼 때, 2022년 기준 등록된 화물차는 369만 6,317대로 전체 자동차 등록대수의 14.5%를 차지하며, 2020년 기준 화물차 일평균 통행량은 470만 8,826대/일로 전년 대비 1.9%가 증가했다.[^2]
이렇듯 통행량이 증가 추세라는 점에서 이동 증가는 곧 온실가스 배출 증가로 이어진다고 볼 수 있다.

> 문제의 핵심은 화물 운송이 화물차에 쏠려 있는 수단 편중으로 인한 화물차의 온실가스 배출량이 증가함에 따라 환경오염을 심각화하고, 건강까지 악화될 수 있는다는 점이다.

- 호남권 물동량 1위는 삼일동(115,016) 이었으나, 반경 10km 내 IC가 0개라 거점 후보에서 자동 제외
- 단순 수치적 물동량만 봤을 때 1위지만, 고속도로 접근성이 없어 **수학적 중심지가 반드시 현실적인 최적의 입지로 이어지지 않음**을 알 수 있음

**3. 민감도 분석 -> 가중치에 흔들리지 않는 안정적 거점을 도출해내고자 함 (99.2% 좌표 정합)**
- 물동량:IC 가중치를 0.5~0.7로 바꾸어가며 민감도 분석 진행
- 영남권을 제외한 모든 권역의 1순위가 동일하게 유지됐고 영남권도 0.6 이상에서 선암동으로 안정되어 **0.65를 채택**
- OD 3,450개 읍/면/동 중 99.2%(3,423개) 가 행정동 경계 좌표와 직접 매칭, 나머지(부천 10 + 미매칭 17)도 보강해 100% 좌표를 확보

## 4. 분석 코드 스니펫 (Code Snippet)

```python
import numpy as np
from sklearn.cluster import KMeans

1) 물동량을 가중치로 한 K-means (K=4) — 좌표로 묶되 물량 많은 동에 비중
X = mart[["lat", "lon"]].values
w = mart["total_vol"].values
km = KMeans(n_clusters=4, random_state=42, n_init=10)
mart["cluster"] = km.fit_predict(X, sample_weight=w)   # 중심점=권역 수요 무게중심(진단용)

# 2) 후보별 반경 10km 내 IC 수 (Haversine)
def haversine(la1, lo1, la2, lo2):
    R = 6371.0
    p1, p2 = np.radians(la1), np.radians(la2)
    dp, dl = np.radians(la2 - la1), np.radians(lo2 - lo1)
    a = np.sin(dp/2)**2 + np.cos(p1)*np.cos(p2)*np.sin(dl/2)**2
    return 2 * R * np.arcsin(np.sqrt(a))

mart["ic_cnt"] = [sum(haversine(r.lat, r.lon, la, lo) <= 10 for _, la, lo in ics)
                  for r in mart.itertuples()]

# 3) 권역 내 정규화 후 스코어 = (0.65*물동량 + 0.35*IC)*100, IC 0개 제외
def norm(s):
    rng = s.max() - s.min()
    return (s - s.min()) / rng if rng else s * 0

for i in range(4):
    g = mart[(mart.cluster == i) & (mart.ic_cnt > 0)]   # IC 0개 동은 거점 후보에서 제외
    score = (0.65 * norm(g["total_vol"]) + 0.35 * norm(g["ic_cnt"])) * 100
    mart.loc[g.index, "score"] = score
```

## 5. 결론 및 기대효과 (Conclusion)
[결론]
- 기존 시군구 단위 분석의 한계를 넘어 미시적 화물차 통행량 분석을 수행.
- 가중 K-Means 로 권역을 구획하고, 권역별 고물동량 읍·면·동을 IC 접근성으로 평가해 **실행 가능한 물류 거점 입지를 검증**하는 실무형 파이프라인 완성.
- 물류 기업에게 배송 리드타임을 최적화할 수 있는 실질적 가이드 제공.
- 공공 및 환경 측면에서 불필요한 도심 진입과 공차 운행을 줄임으로써, **도로 혼잡과 탄소 배출 저감에 기여**할 것으로 예상됨.

[한계점]
- 한국도로공사 API 제약으로 인해 실제 도로 주행거리가 아닌 직선거리 반경 10km 기준으로 1차 필터링을 진행함.
- 화물차의 톤수나 적재 품목의 특성을 세분화하지 않고 전체 확장 통행량 총합을 대상으로 가중치를 적용함.
- 시간 데이터를 반영하지 않아, 주간/야간 세션의 분리가 없어 일 평균 통행량 기준의 시간대별 가변적 통행 특성을 반영하지 못함.

[기대효과]
1) 기업 측면: 읍면동 단위의 정밀한 입지 제안 가능
-  물류 거점 신설 시 발생할 수 있는 시행착오 비용을 최소화하고 배송 리드타임을 최적화

2) 공공/환경 측면: IC연계 거점 배치를 통한 녹색 물류 실현 가능
-  실제 고속도로 IC 연계성을 고려한 거점 배치로 화물차의 불필요한 공차 운행 거리와 도심 진입을 줄여, 도로 혼잡 완화와 탄소 배출 저감(ESG)에 기여
